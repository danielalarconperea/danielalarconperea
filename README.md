## Perfil técnico

Me estoy formando en administración de sistemas, redes, scripting y seguridad. En este GitHub mantengo mi portfolio de ASIR ordenado por áreas, con prácticas reales de clase, laboratorios personales y el proyecto final del ciclo.

Me interesa especialmente la parte donde se juntan infraestructura, automatización y seguridad: servicios Linux, redes, hardening, monitorización, despliegues con Docker y pequeños scripts que ayudan a administrar sistemas de forma más limpia.

## Áreas que trabajo

| Área | Tecnologías y conceptos |
| --- | --- |
| Sistemas | Linux, Bash, PowerShell, permisos, procesos, cron, servicios y almacenamiento. |
| Redes | Routing, ACL, OSPF, Packet Tracer, servicios de red y troubleshooting. |
| Automatización | Python, scripting, tareas administrativas y pruebas de backend. |
| Bases de datos | SQL, MySQL, procedimientos, triggers y administración básica. |
| Despliegue web | Apache, Nginx, PHP, MySQL, Docker y WordPress. |
| Seguridad | Hardening, monitorización, MQTT/mTLS, análisis de logs y respuesta ante eventos. |
